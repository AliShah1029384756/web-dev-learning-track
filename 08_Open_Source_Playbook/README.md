# Open Source Playbook

## Goal
Build consistent, visible, and meaningful contributions that improve both community impact and GitHub contribution graph.

## Weekly System

1. Pick 2 repositories aligned with your stack.
2. Submit 3 small contributions (docs/tests/fixes).
3. Submit 1 medium contribution (feature/refactor).
4. Record merged outcomes in contribution tracker.

## Contribution Types
- Documentation improvements
- Bug reproduction and fixes
- Test coverage additions
- Starter issues and triage support
