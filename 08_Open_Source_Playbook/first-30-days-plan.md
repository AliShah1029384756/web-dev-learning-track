# First 30 Days Open Source Plan

## Week 1
- Set up contribution-friendly repositories list
- Read contributing guides and code style

## Week 2
- Submit docs and test improvements
- Build review habit for maintainers' feedback

## Week 3
- Submit bug fix and one refactor PR
- Improve issue quality with repro steps

## Week 4
- Submit one medium feature contribution
- Publish monthly summary in profile docs
