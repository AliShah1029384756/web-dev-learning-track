# Web Dev Learning Track

A structured, self-guided roadmap to help learners move from beginner fundamentals to practical full-stack readiness.

## Who This Is For

- Beginners starting web development from scratch
- CS students who want practical project experience
- Self-learners building a strong GitHub portfolio
- Learners preparing for internships and entry-level roles

## How To Use This Track

1. Follow the weeks in sequence (do not skip fundamentals).
2. Complete the weekly goals and project tasks.
3. Push your work to GitHub with clean commit messages.
4. Review mistakes and improve your previous projects weekly.
5. Repeat weak topics before moving to the next level.

## 12-Week Plan

- [ ] Week 1 complete - HTML basics, semantic structure, simple page layout
- [ ] Week 2 complete - CSS fundamentals, Flexbox, responsive design basics
- [ ] Week 3 complete - JavaScript basics (variables, functions, conditions)
- [ ] Week 4 complete - DOM manipulation and events (build: Calculator)
- [ ] Week 5 complete - Arrays, objects, localStorage (build: Todo App)
- [ ] Week 6 complete - CRUD logic and state updates (build: Notes App)
- [ ] Week 7 complete - Timers, scoring logic, UI feedback (build: Quiz App)
- [ ] Week 8 complete - Data calculations and rendering (build: Expense Tracker)
- [ ] Week 9 complete - Portfolio polish and project presentation
- [ ] Week 10 complete - API integration with fetch/async-await (build: Weather App)
- [ ] Week 11 complete - Backend fundamentals with Node.js and Express
- [ ] Week 12 complete - Build and document CRUD REST API project

## Outcome After Completion

By the end of this track, you should be able to:

- Build and deploy beginner-to-intermediate web projects
- Write clean, organized HTML/CSS/JavaScript code
- Work with APIs and asynchronous JavaScript
- Build a basic backend using Node.js and Express
- Maintain professional project documentation
- Present a portfolio suitable for internship/job applications
